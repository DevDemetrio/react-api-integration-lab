# React API Integration Lab

This project was created to practice API integration using a modern React architecture.  
The application consumes the public API https://restful-api.dev and focuses on organizing the frontend using a service layer, schema validation and data fetching best practices.

## 🚀 Technologies

- React
- TypeScript
- Axios
- React Query (TanStack Query)
- Zod (Schema validation)

## 📚 Purpose

The goal of this project is to improve understanding of:

- API consumption in React
- Service layer pattern
- Data validation with Zod
- Handling API responses
- React Query for data fetching and caching
- Type-safe frontend architecture with TypeScript
